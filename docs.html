Avatars

Avatar are images which can be used to show user's profile image. Use base class .avatar to generate a avatar.

Image Avatars

Image avatars can be created with class .avatar-img

Letter Avatars

Letter avatars can be created with class .avatar-letter. Add the color classes to apply color to the avatars. This can be used with maximum of 2 letters.

Different Sizes

Different sizes are available for the avatars, classes - .avatar-sm, .avatar-md, .avatar-lg

Icon Avatars

Icon avatars have class .avatar-icon. You can use svgs and icons from font awesome.

Variants - Square

Adding a class .square to existing class, will generate a square shaped avatar.

Installation

To start using the components in your project import the stylesheet in your css or html file.

Customization

By Default these reset styles will be applied to your project when you import Bubble UI.

You can also add your own classes to customize the components as per your need.

Typography

Default Font is set to Roboto. Has 6 font weights - 100; 300; 400; 500; 700; 900

For heading u can use h1, h2, h3, h4, h5, h6 elements. The same font-styling is present for class names h1, h2, h3, h4, h5, h6 class . You can add one of these classes to style the text.

You can add the color classes to change the colour of text. Classes are available for all the variants.

Avatars

Avatar are images which can be used to show user's profile image. Use base class .avatar to generate a avatar.

Image Avatars

Image avatars can be created with class .avatar-img

Letter Avatars

Letter avatars can be created with class .avatar-letter. Add the color classes to apply color to the avatars. This can be used with maximum of 2 letters.

Different Sizes

Different sizes are available for the avatars, classes - .avatar-sm, .avatar-md, .avatar-lg

Icon Avatars

Icon avatars have class .avatar-icon. You can use svgs and icons from font awesome.

Variants - Square

Adding a class .square to existing class, will generate a square shaped avatar.

Alerts

Bubble has 4 types of Alerts - success, warning, info and warning. To get the icons use the font awesome cdn. Icons are attached to special classes. .alert is the base class.

Basic Alerts

Add .alert-error, .alert-info, .alert-warning, .alert-success along with .alert to get the basic variants.
